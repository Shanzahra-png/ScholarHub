**Final Year Project (BS IT)**

A web-based Scholarship Management System developed to help students search and apply for national and international scholarships from one platform.

**Features**
- Student Registration & Login
- Scholarship Listings
- Domestic Scholarships
- International Scholarships
- Search Functionality
- News & Updates
- Contact Page
- Responsive Design
**Technologies Used**
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Bootstrap
